# 概述

### YJMSQL 介绍

YJMSQL 是一门 SQL 的方言，常见于某位高校教师的日常教学和编纂书籍当中。

这位老师说过主要学思想，所以 YJMSQL 与其他 DBMS 系统的语法可能有较大差异；而他又说过错误的语法会对分数造成很大的影响，故本人认为本系列 Wiki 有其存在的必要。
